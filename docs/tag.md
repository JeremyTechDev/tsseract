## **Tag Endpoint**

The tag endpoint handles the searches of the posts' tags by their name.

- **URL**

  ```
  /api/tags
  ```

| Gist:                                     | Route: | Method: | Success Status: | Type:   |
| ----------------------------------------- | ------ | ------- | --------------- | ------- |
| Returns the data contained the user token | `/`    | `GET`   | 200             | PRIVATE |
